# Twitch VOD & Clip Downloader

Расширение для Яндекс Браузера (и любого Chromium-браузера, Manifest V3): скачивает записи стримов (VOD) и клипы с twitch.tv по кнопке на странице, с выбором качества и возможностью скачать только фрагмент VOD.

## Установка

1. Откройте страницу управления расширениями своего браузера:
   - Яндекс Браузер — `browser://extensions`
   - Chrome / Chromium / Brave / Vivaldi — `chrome://extensions`
   - Microsoft Edge — `edge://extensions`
   - Opera — `opera://extensions`
2. Включите переключатель **«Режим разработчика»** (в большинстве браузеров — справа вверху, в Edge — в левой колонке).
3. Нажмите **«Загрузить распакованное расширение»** и укажите папку `twitch-vod-downloader`.
4. Откройте любую страницу VOD (`twitch.tv/videos/…`) или клипа — в правом верхнем углу плеера появится кнопка **«⬇ Скачать»**.

Сборка не требуется: все файлы, включая библиотеку mux.js, лежат в папке расширения.

## Использование

- **Клип**: нажмите кнопку, выберите качество, нажмите «Скачать» — файл mp4 уйдёт в обычные загрузки браузера.
- **VOD**: нажмите кнопку, выберите качество и при желании отрезок «От»/«До» (формат `1:23:45`; пустое «От» = с начала, пустое «До» = до конца). После нажатия «Скачать» откроется вкладка менеджера загрузки: выберите, куда сохранить файл, — и загрузка начнётся. Вкладку менеджера нельзя закрывать до завершения; остальным браузером можно пользоваться свободно.

VOD сохраняется как fragmented MP4 (H.264 + AAC без перекодирования): сегменты HLS скачиваются параллельно, на лету перепаковываются из MPEG-TS в MP4 (mux.js) и порционно пишутся на диск, поэтому потребление памяти ограничено даже на многочасовых записях.

## Ограничения

- **Максимум Full HD без логина.** Twitch выдаёт анонимным запросам качество не выше 1080p (примерно 12500 кбит/с). Варианты 1440p/4K в списке качеств не появятся, даже если стример их писал.
- **Саб-онли VOD не поддерживаются** — расширение работает без авторизации и покажет понятную ошибку.
- **Фрагмент режется по границам сегментов HLS** (обычно 10–13 секунд), поэтому фактические границы могут отличаться от заданных примерно на ±10 секунд.
- **Client-ID Twitch периодически меняется.** Расширение при ошибке само достаёт актуальный из главной страницы Twitch и кэширует его, так что обычно это незаметно.
- Если в VOD были вставки/склейки (`EXT-X-DISCONTINUITY`), в месте склейки возможен небольшой скачок таймстемпов.
- Скачивание чужого контента с Twitch формально противоречит условиям сервиса — используйте для личных целей.

## Структура

- `manifest.json` — Manifest V3, разрешения и content script.
- `background.js` — service worker: запросы к GQL/usher, загрузка клипов, открытие менеджера.
- `content.js` / `content.css` — кнопка и панель выбора качества на странице Twitch.
- `manager.html` / `manager.js` — вкладка-менеджер загрузки VOD (скачивание, транс-мукс, запись на диск).
- `lib/mux.min.js` — mux.js 7.1.0 (Apache-2.0), перепаковка MPEG-TS → fMP4.
